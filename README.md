# AI Document Intelligence

Full-stack **document Q&A** app: upload PDF/DOCX, chunk + embed with `pgvector`, then ask questions with page citations, extract structured data, or compare two documents.

**Source code (clone this path in the repo):** [`c-projects/AI-DOC-INTEL/ai-doc-intel/`](c-projects/AI-DOC-INTEL/ai-doc-intel/)

**Setup, architecture, and screenshots:** [Project README](c-projects/AI-DOC-INTEL/ai-doc-intel/README.md)

**Stack:** FastAPI · Next.js · Supabase (Auth, Postgres, Storage) · OpenAI-compatible embeddings & chat
